# Forge Digital Landing Page

Beautiful, animated landing page for Forge Digital — a software studio building landing pages, automations, and SaaS MVPs.

## Features

- **Dark theme** with industrial luxury aesthetics
- **Rich animations** throughout: scroll reveals, parallax, 3D card tilt, shimmer effects
- **Fully responsive** design
- **React-powered** with inline components
- **Customizable** via tweaks panel (accent color, hero headline, button style)
- **Conversion-focused** copy and clear CTAs

## Tech Stack

- React 18 (via CDN)
- Vanilla CSS with CSS variables for theming
- Babel for JSX transpilation in the browser

## Sections

1. **Hero** — Animated headline with stats strip
2. **Trust Bar** — Key differentiators
3. **Services** — 6 service offerings with icon animations
4. **How It Works** — 3-step process with draw-in connector
5. **Portfolio** — Case study showcase
6. **Pricing** — 3-tier pricing model
7. **FAQ** — Expandable Q&A
8. **Final CTA** — Conversion section with ambient glow
9. **Footer** — Links and copyright

## Animations

- Scroll progress bar
- Cursor glow tracking
- Magnetic button interactions
- Typewriter text reveal
- Text scramble on section labels
- Word-by-word title reveals
- 3D card tilt on hover
- Shimmer sweep on CTAs
- Draw-in line connector
- Parallax hero background
- Icon pop animations
- Pulsing featured pricing card

## Customization

The tweaks panel (bottom-right) allows you to:
- Change accent color (Amber, Teal, Violet)
- Swap hero headline variants
- Adjust button border radius

## Getting Started

Simply open `project/index.html` in a modern browser. No build step required.

## Colors

- **Dark bg**: `#0e0c0a`
- **Light fg**: `#f2ede6`
- **Accent (Amber)**: `oklch(0.72 0.16 52)`
- **Steel (Blue)**: `oklch(0.72 0.06 220)`

## Typography

- **Headlines**: Syne (400, 600, 700, 800)
- **Body**: DM Sans (300, 400, 500)
